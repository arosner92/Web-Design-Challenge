Summary: Latitude versus X

This project was for analyzing weather changes as you approach the equator. To do the analysis we pulled data from the OpenWeatherMap API to create a dataset of cities.

We used Matplotlib to plot variables of the weather vs. latitude. This site provides the source data and some examples of visualizations with explanations and descriptions of trends and correlations.
